# re-outreach-scripts

Real estate investor outreach scripts built and refined by Lyn (Lynxify).
Covers Facebook Group deal sourcing, Realtor partnerships, motivated seller
outreach, and co-investor lead generation.

These aren't copy-paste blasts. Every script here is built around one rule:
**lead with empathy, close with clarity.**

---

## Who this is for

- Wholesalers sourcing off-market deals via social media
- Fix-and-flip investors building Realtor referral pipelines
- Real estate entrepreneurs looking for passive capital partners
- Marketing specialists managing outreach for RE clients

---

## Folder structure

| Folder | What's inside |
|---|---|
| `01-facebook-group-scripts/` | Comment + DM scripts for FB Group deal sourcing |
| `02-realtor-outreach/` | Cold outreach + partnership pitches for agents |
| `03-motivated-seller/` | Empathy-first DMs + organic content hooks |
| `04-co-investor-outreach/` | Intro scripts for passive investor sourcing |

---

## How to use these

1. Pick the script folder that matches your goal
2. Read the **context note** at the top of each file
3. Personalize the `[brackets]` — never send a template cold
4. Test 2–3 variations and track your reply rate
5. If it's not working in 50 sends, the targeting is wrong — not the script
